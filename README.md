#### 项目介绍
* xmopay支付系统， 使用主流语言Java开发，用spring-cloud微服务架构开发，各个业务功能模块分离，整体架构松耦合和可扩展。
目前，已接入微信、支付宝等主流支付渠道，可直接用于生产环境。
